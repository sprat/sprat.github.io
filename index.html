Open-Source Projects

Here are my open-source projects:

  • albumart is a command-line album art downloader tool. This tool recursively scan the root folder of your music library in order to find audio files, extracts their metadata (ID tags, etc.) and downloads album cover art from some famous cover art web sites according to the metadata.
  • minify provides distutils commands for minifying CSS and JS resources using the well-known YUI compressor from Yahoo! Inc.
  • web.recipes is a collection of recipes that solve specific HTML/CSS design problems.
  • yaak.inject provides dependency injection to your applications.
